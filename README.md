# Movie Sentiment Analysis

A simple Movie Sentiment Analysis web application built using **Python** and **Streamlit**. The application predicts whether a movie review is **Positive** or **Negative** and highlights the important keywords that influenced the prediction.

## Features

* Predicts sentiment of movie reviews
* Displays Positive or Negative result
* Shows matched keywords/phrases
* Interactive and easy-to-use interface

## Technologies Used

* Python
* Streamlit
* HTML
* JavaScript
* Jupyter Notebook

## How to Run

1. Clone the repository:

```
git clone https://github.com/Withansh/Movie-Sentiment-Analysis.git
```

2. Navigate to the project folder:

```
cd Movie-Sentiment-Analysis
```

3. Install the required packages:

```
pip install streamlit pandas scikit-learn
```

4. Run the application:

```
streamlit run app.py
```
